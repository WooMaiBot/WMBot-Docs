# 货币兑换

查询汇率并在不同种类的货币间转换。

## Usage

``` 
/exchange [金额] <起源币种> <目标币种>
```

## Example

```
/exchange USD CNY  // 1 USD 等于多少 CNY
/exchange 20 EUR JPY   // 20 EUR 等于多少 JPY
```
