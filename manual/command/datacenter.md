# Telegram 数据中心

查询自己或他人的 Telegram 账户所在数据中心。

## Usage

```
/dc [用户名]
```

如不指定用户名，则查询自己。

> 自 WooMai Bot V3 起，可以正确识别 DC2/3 用户。
